
#  **TravelX – Explore The World**

TravelX is a modern, responsive **landing page** designed for travel websites. It highlights popular destinations, upcoming travel packages, and essential contact details with a clean UI and smooth user interactions. The project is built using HTML, CSS, and JavaScript and works seamlessly across all screen sizes.

---

##  **Features**

* **Responsive Layout** – Adjusts smoothly on mobile, tablet, and desktop
* **Interactive Navigation** – Navbar style changes on scroll
* **Hero Section** – Clean and attractive banner with a call-to-action button
* **Top Destinations** – Grid-style layout with subtle hover effects
* **Exclusive Packages Section** – Space reserved for future travel deals
* **Smooth Scrolling** – Clicking navigation links smoothly scrolls to sections
* **Simple JS Interactions** – Hamburger menu for mobile view and scroll effects

---

##  **Technologies Used**

* **HTML5** – Structure and layout
* **CSS3** – Styling, design, responsiveness
* **JavaScript** – Interactivity and UI behavior

---

##  **How to Run**

1. Download or clone the project folder
2. Open the file **index.html** in any browser
3. Explore the sections and UI elements

> This is a static project — no server is required.
